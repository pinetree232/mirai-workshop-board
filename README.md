# 未来から逆算しよう｜ワーク① ボード

合宿ワーク用の単一ページWebアプリ（HTML 1ファイル）。参加者の回答を貼り付け、Claude APIで①「2030年の社会の変化」と②「現場で変わってほしい点」をそれぞれ3グループに自動分類し、未来と現場のギャップを要約表示します。

## 使い方

1. `index.html` をブラウザで開く（ダブルクリックでOK）。
2. 上部の欄に Anthropic APIキー（`sk-ant-...`）を入力する。
   - キーはこの画面に入力するだけで、リポジトリには保存しないでください。
3. `【名前】①…②…` の形式で全員分（最大9名）の回答を貼り付け、「読み込む」を押す。
   - 「サンプルを入れる（9名）」で動作確認用のデータを投入できます。
4. 「グルーピング実行」を押すと、3グループへの分類とギャップ要約が表示されます。

## 技術

- React 18（UMD / CDN）+ Babel Standalone
- Tailwind CSS（CDN）
- Anthropic Messages API（`claude-sonnet-4-6`）をブラウザから直接呼び出し

## 注意

- APIキーはコードや本リポジトリにコミットしないでください。実行時に画面で入力します。
- ブラウザから直接APIを呼ぶため、`anthropic-dangerous-direct-browser-access` を有効化しています。社内・合宿などクローズドな利用を想定しています。
